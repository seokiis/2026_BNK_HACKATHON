## 상/중/하 등급 판정 기준

각 항목은 규정·심사 기준 대비 평가하여 등급을 부여한다.

- **상**: 기준을 충분히 충족, 리스크 낮음, 근거 명확
- **중**: 부분 충족, 일부 리스크·불확실성 존재, 보강 여지 있음
- **하**: 기준 미달 또는 중대 리스크, 근거 취약

> 중·하 등급은 UI에서 마우스 오버 시 "위험 상태 보강" 버튼으로 전환되어,
> 외부 데이터 추적관이 외부자료로 상태 보강 워딩을 생성한다 (기능명세 5번).

### 등급 산정 시 고려
- 항목별로 규정에서 정한 임계값(예: 부채비율, DSCR)을 우선 적용.
- 임계값이 모호하면 배정 심사역의 과거 승인 사례를 RAG로 참조해 판단.
- 등급에는 항상 근거를 명시한다 ("DSCR 1.1배로 기준 1.25 미달 → 하").

## 데이터 부족 / 누락 판정 기준

등급과 별개로, 의견 작성의 근거 데이터 상태를 진단한다.

- **데이터 부족**: 관련 데이터가 있으나 판단에 불충분 (예: 1개년 재무만 존재)
- **데이터 누락**: 필요한 데이터가 아예 없음 (예: 매출자료 미제출)

> 부족·누락 라벨은 UI에서 마우스 오버 시 "데이터 보강" 버튼으로 전환되어,
> 외부 데이터 추적관이 외부자료로 초안을 보강한다 (기능명세 6번).

### 판정 시 주의
- 데이터가 없으면 수치를 추정·생성하지 말고 "누락"으로 표시한다.
- 어떤 데이터가 있어야 했는지(예: "최근 3개년 재무제표 필요")를 함께 안내한다.
