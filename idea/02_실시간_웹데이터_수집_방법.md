## 1. 사용 가능한 방법 (Azure 제약 내)

### ⭐ 방법 A: Azure OpenAI — Bing Grounding (추천)

Foundry/Azure OpenAI에서 GPT 호출 시 `tools`에 `bing_grounding`을 추가하면,
**GPT가 직접 Bing 검색을 수행**하고 결과를 참고해 답변한다.

```python
# 백엔드 코드 예시
from openai import AzureOpenAI

client = AzureOpenAI(...)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{
        "role": "user",
        "content": "반도체 소재 산업의 2025-2026년 현황과 전망을 요약해줘. BSI지수, 수출증감률, 주요 뉴스를 포함해."
    }],
    extra_body={
        "data_sources": [{
            "type": "bing_grounding",
            "parameters": {
                "connection_id": "<Azure AI connection resource ID>",
                "market": "ko-KR",
                "count": 10
            }
        }]
    }
)
# → GPT가 Bing 검색 후 결과 + 출처 URL 포함해서 응답
```

**장점:**
- API 한 번 호출로 검색 + 요약 + 출처 모두 해결
- Azure 기술 활용 → 평가 점수 ↑
- 별도 Bing API 구독 불필요 (Foundry 연결로 가능)

**필요한 설정:**
1. Azure AI Foundry에서 Bing Search 연결(Connection) 생성
2. 해당 connection ID를 백엔드 환경변수에 등록
3. 호출 시 `data_sources`에 포함

---

### 방법 B: Bing Search API 직접 호출

별도 Azure Cognitive Services > Bing Search 리소스 생성 후 직접 호출:

```python
import requests

headers = {"Ocp-Apim-Subscription-Key": BING_API_KEY}

# 뉴스 검색
news = requests.get(
    "https://api.bing.microsoft.com/v7.0/news/search",
    headers=headers,
    params={"q": "반도체 소재 산업 전망 2026", "mkt": "ko-KR", "count": 5}
).json()

# 웹 검색 (산업 보고서)
web = requests.get(
    "https://api.bing.microsoft.com/v7.0/search",
    headers=headers,
    params={"q": "반도체 BSI지수 2025 site:bok.or.kr OR site:kiet.re.kr", "mkt": "ko-KR"}
).json()
```

**장점:** 원시 검색 결과를 직접 제어 가능  
**단점:** 별도 리소스 생성 필요, 결과 파싱·요약 로직 별도 작성

### 방법 C: 공공 API (무료, 보조 수단)

| API | 데이터 | URL |
|-----|--------|-----|
| DART OpenAPI | 기업 공시 (재무제표, 사업보고서) | https://opendart.fss.or.kr |
| 한국은행 ECOS | BSI, 경기전망지수 | https://ecos.bok.or.kr/api |
| 통계청 KOSIS | 산업생산지수, 수출입 | https://kosis.kr/openapi |
| 산업연구원 | 산업 동향 리포트 | (크롤링 필요) |

**용도:** 정량 수치 뒷받침 (BSI 118 같은 구체적 숫자)  
**제약:** 해커톤 시간 내 연동 복잡 → 방법 A의 보조로만 사용

## 2. 추천 아키텍처

```
[재무 분석 → 반려 판정]
        │
        ▼
[산업현황 보강 트리거]
        │
        ├─── (1) Bing Grounding으로 산업 동향 검색
        │         "업종코드 C26 → 반도체 → 최근 산업현황 전망"
        │
        ├─── (2) Bing Grounding으로 관련 뉴스 검색  
        │         "반도체 소재 기업 투자 확대 뉴스"
        │
        └─── (3) 결과를 7명 토론의 "산업분석가" agent에게 주입
                  → 토론에서 재무 불리 vs 산업 호황 논쟁

[최종 심사의견서]
  ├── 재무 위험: 부채비율 250% (내규 초과)
  ├── 산업 보완: 반도체 산업 호황(BSI 118), AI 수요 확대
  ├── 뉴스 근거: "삼성전자 반도체 소재 투자 3조 확대" (출처 URL)
  └── 종합: 조건부 승인 검토 권고
```

## 4. 해커톤에서 실제로 할 일

1. **Azure AI Foundry에서 Bing Search Connection 생성** (포털에서 5분)
2. **백엔드에 `/api/industry-analysis` 엔드포인트 추가**
   - 입력: 업종코드, 기업명
   - 내부: Bing Grounding으로 GPT 호출
   - 출력: { status, trends, news[], sources[] }
3. **7명 토론 오케스트레이션에서 산업분석가 agent에게 결과 주입**
4. **프론트: "산업분석" 탭에 결과 표시** (출처 URL 링크 포함)

## 5. 프롬프트 예시 (Bing Grounding용)

```
당신은 BNK금융그룹의 산업 분석 전문가입니다.
다음 기업의 업종에 대해 최신 산업현황을 조사하세요.

기업명: {company_name}
업종: {industry_name} (코드: {industry_code})

다음 항목을 조사해 JSON으로 응답하세요:
1. 산업 경기 상태 (호황/보통/침체)
2. BSI 또는 경기전망지수 (최근 발표 기준)
3. 수출입 증감률 (YoY)
4. 최근 3개월 주요 뉴스 3건 (제목, 요약, 출처URL)
5. 향후 1년 전망 요약
6. 해당 산업이 대출 상환 능력에 미치는 긍정/부정 영향 판단
```
